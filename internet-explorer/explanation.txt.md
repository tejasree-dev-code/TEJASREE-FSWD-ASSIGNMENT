### Client-Server Architecture

##### 1\. Client-Server Model:

Client: Requests data (e.g., browser).

Server: Provides data (e.g., website host).

##### 2\. Steps When Opening a Website:

1.Enter URL → browser sends request.

2.DNS converts domain to IP.

3.Browser sends HTTP request to server.

4.Server processes request and sends response.

5.Browser renders the page
