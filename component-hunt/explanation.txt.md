#### Website chosen: https://www.youtube.com

###### 1\. Navigation Header (Global)

Purpose: Provides constant access to search and user settings.

Elements: Logo, Search Bar (with voice search), Create/Upload button, Notifications, User Profile menu

###### 2\. Sidebar (Primary Navigation)

Purpose: Main directory for library and subscriptions

Elements: Home, Shorts, Subscriptions, Library links (History, Watch Later), Subscribed Channels list

###### 3\. Filter Bar (Categorization Chips)

Purpose: Refine content feed by category

Elements: Scrollable horizontal buttons (Music, Gaming, Live, etc.)

###### 4\. Video Feed (Content Grid)

Purpose: Main discovery area

Elements: Grid of Video Cards – thumbnail, duration, channel avatar, title, views/date

###### 5\. Watch Page (Secondary Layout)

Purpose: Focused viewing and engagement

Elements: Video Player, Description, Comments, "Up Next" Sidebar.
